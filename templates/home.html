Problem Statment
The development of this system involves several critical steps. First, a custom image dataset of 4 to 5 item categories is created and annotated for training the YOLOv10 model. The model is then trained to achieve optimal accuracy in detecting these items. A Flask-based web application serves as the user interface, where customers can upload images of their shopping items. Upon receiving an image, the system preprocesses it and uses the YOLOv10 model to detect and classify the items. The detected items are cross-referenced with a database to retrieve their prices, and the total bill is calculated and displayed to the user. For this prototype, the payment process is simulated by displaying a payment success page, without integrating a real payment gateway.
Introduction
In traditional shopping environments, customers often face long queues at billing counters, leading to inconvenience and reduced customer satisfaction. Moreover, manual billing is prone to human errors, resulting in incorrect billing and customer dissatisfaction. The proposed self-checkout system addresses these issues by allowing customers to take a single photograph of all their shopping items using a mobile phone, which is then uploaded to a web application developed using Flask. The uploaded image is processed by the YOLOv10 model, which identifies all the items in the image and calculates the total bill based on predefined prices.
Proposed Solution
The 'Self Checkout for Shopping Mall using YOLO Object Detection' project aims to revolutionize the traditional shopping experience by introducing an automated, user-friendly self-checkout system. This project leverages the power of computer vision and machine learning to streamline the billing process, reduce wait times, and enhance customer satisfaction. The system is designed to detect and identify shopping items using the YOLO (You Only Look Once) object detection model, specifically the YOLOv10 variant, known for its high accuracy and real-time detection capabilities.
Conclusion
The "Self Checkout for Shopping Mall using YOLO Object Detection" project successfully integrates advanced technologies to enhance the retail shopping experience. By leveraging the YOLOv10 object detection model, the system provides accurate and efficient recognition of multiple items in a single image, significantly reducing the time and effort required for traditional checkout processes. The experimental results indicate high accuracy in object detection, with an overall accuracy rate of approximately 89.74%. The billing calculations were also highly accurate, achieving a 96% success rate, which underscores the system's reliability. User acceptance testing revealed an encouraging satisfaction rate of over 83%, demonstrating that users found the system intuitive and beneficial.
